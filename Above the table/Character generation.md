# Getting started
Characters are created using the rules in the Dolmenwood Player’s Book. Ask me for access to it.

You can print out a blank character sheet from here: https://www.drivethrurpg.com/en/product/497368/dolmenwood-character-sheet

You can generate a random character to get a feel for different parts of character creation here 
- https://www.dolmenwood.necroticgnome.com/characters/
- However it doesn’t offer much in the way of customisation 

# Main procedure
Follow the guidance in the Player’s book, which starts on page 18.

Some common differences with other systems
- You roll 3d6 for each stat *in order*
- You can roll **two sets of stats** and choose the array that better fits
- Embrace the fickleness of the dice to generate interesting characters!
- Note that you can adjust ability scores according to the rules in Step 4 on p18
- You can also ditch an array if it has 6 or less in two or more stats, or nothing above 8 in all
